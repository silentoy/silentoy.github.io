Github + Hexo博客系统的搭建

发表于 2015-06-25   |   分类于 技巧分享   |  

自认为不是一个爱写东西的人,主要是不会写.但是最近在慢慢培养自己的一些习惯,比如从上个月开始的晨跑,已经突破100KM了.这个月开始的每日一拍也已经十几天了.

慢慢的发现偶尔要求自己去做些事情是挺好的,那就开个博客吧,每周写上一两篇,或者发一两张我拍的照片,都是不错的存在

使用什么搭系统

要写东西,得有个平台,作为一个资深码农,显然不应该用商业博客来做,那样显得会很LOW. 那只有一个选择了, 买域名, 开空间, 挂Wordpress, 这是好几年前的套路,现在都用Github+Hexo,高端大气上档次,那怎么弄呢?

搭建步骤

网上已经有了很多现成的教程,这几个是比较详细的,可以留作参考

  • http://www.aips.me/hexo-independent-blog-new-ways.html
  • https://hexo.io/zh-cn/
  • https://pages.github.com/

常用命令

  • git add —all
  • git commit -m ‘comment’

  • git push origin master

  • hexo n “标题” 写文章

  • hexo g -d 发布站点
